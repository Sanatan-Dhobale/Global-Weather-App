#🌍 Global Weather App

A simple and responsive Global Weather App built with HTML, CSS, and JavaScript, allowing users to fetch and display real-time weather information for any city worldwide using a weather API.

This project demonstrates core web development skills, including API integration, DOM manipulation, asynchronous JavaScript, and responsive UI design — perfect for beginner to intermediate frontend developers.

Live Demo: https://global-weather-app-mu.vercel.app/

#📌 Features

✅ Search weather by city name
✅ Display current temperature, humidity, wind speed, and more
✅ Fetch live data from a public weather API (e.g., OpenWeatherMap)
✅ Responsive design that works on desktop and mobile
✅ Clean UI with interactive results and weather icons

#🛠️ Technologies Used

✔ HTML5
✔ CSS3 (Responsive layout)
✔ JavaScript (Fetch API + async/await)
✔ Weather API integration (e.g., OpenWeatherMap)

#🚀 How It Works

User enters a city name in the search field.

The app sends a request to a weather API for real-time data.

Weather data is processed and displayed on the screen (temperature, conditions, humidity, etc.).

The UI updates dynamically without a page reload.
